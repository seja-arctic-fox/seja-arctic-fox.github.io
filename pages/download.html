    Building

  • Download the source code

    First start by downloading the source code by clicking the button above. It has been tested on Linux. You're free to edit the code to work on any other platform you use.

    Or you can run: 

    git
     clone https://gitlab.fjfi.cvut.cz/sellnmic/vidcom-gui

  • After downloading (and extracting)

    Go into the root directory of the project and run this command: 

    meson
     setup build

    After that, go in the newly created directory "build" and run this command to compile: 

    meson
     compile

    If you want to compile and install in your system at the same time, you can run: 

    meson
     install
